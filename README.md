# haskell ray marcher
*A ray marcher in Haskell, up to be implemented.*

### Requirements:
    - Stack version: 2.1.3
    - hpack version: 0.31.2

### How to build:
```sh
$ stack build
```

### How to run:
```sh
$ stack run
```
